# 👟 E-commerce Shoe Store

A full-stack e-commerce web application for selling shoes, built with modern technologies.
---

## 🚀 Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS

### Backend
- Node.js (Express.js)
- RESTful API
- JWT Authentication

### Database
- MongoDB

### DevOps
- Docker (multi-container setup)
---

## ✨ Features

### User
- Register / Login (JWT Authentication)
- Browse products
- Add to cart
- Place orders

### Admin
- Manage products
- Manage orders
- Dashboard overview

---

## 🐳 Run with Docker

```bash
docker-compose up --build
```
